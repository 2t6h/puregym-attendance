# puregym-attendance

puregym-attendance is a Python client to query the PureGym Mobile API for live gym attendance statistics.

## Usage

```python
puregym.py [email] [pin]

Optional Arguments
--gym [gym name OR gym ID] (defaults to puregym home gym)
```
